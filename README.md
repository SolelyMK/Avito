# Avito
Тестовое задание для авито


Для запуска проекта необходимо
В первую очередь скачать папку baclend. Открыть её в редакторе кода. Запустить команду npm i
Далее запустить команду npm start

Затем скачать папку frontend. Открыть её в редакторе кода. Запустить команду npm i
Далее запустить команду npm run dev

(В случае проблем с запуском из-за ограничений на каждом шаге стоит вместо npm использовать npm.cmd)

Подключить LLM не удалось, поэтому инструкцию по запуску не прилагаю. Тем не менее прошу проверить работу.

Собственные решения или особенности:
- Активация фильтров и сортировок осуществляется по кнопке поиска у строки поиска. Это сделано для того, чтобы не отправлялись лишнии запросы, если параметров несколько
- Сортировки по цене скрыты для выбора в выпадающем списке, поскольку нет вызможности их высвать через запросы
- Отсутствуют переходы по нажатию на объявление, поскольку backend не предоставляет id
